# Платформа ИСККОН (iskcone.com)

Профессиональный мультиязычный сайт и приложение, объединяющие всю информацию и
сервисы ИСККОН: поиск храмов и расписания, философию, книги, энциклопедию личностей,
прасадам, Food for Life, пожертвования, образование, медиа и календарь Вайшнавов.

**Архитектурная модель — франшиза / мульти-тенант:** глобальный контент создаётся
один раз, а каждый храм подключается как франчайзи и наследует его, добавляя только
локальные данные (расписание, контакты, божества, события, пожертвования).
