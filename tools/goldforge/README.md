# Кузница золота (goldforge)

Конвейер сборки **авторитетной мини-книги** о личности — по закону ЗКН-Р003
(золото = исчерпание источников) и ЗКН-Р004 (уровень вычисляется гейтом).

## Зачем

Прежняя жатва искала по ОДНОЙ форме имени (русской), а первоисточники лежат
на английском. Для Нароттамы дас Тхакура это означало:

| | было (RU-grep) | стало (кузница) |
|---|---|---|
| упоминаний | 411 | **1 588** |
| источников | 6 | **11** |
| материала | — | **949 257 симв.** |
| карточка | 12 849 симв. | — |
| **покрытие** | — | **7,2 %** |

Две главные книги о нём — `Narottama-Vilasa.EN` (524 упоминания) и
`Bhakti-Ratnakara.EN` (426) — не находились вообще.

## Стадии

```sh
# 0 · ПАСПОРТ — все формы имени (RU + склонения + EN + IAST + алиасы)
python3 tools/goldforge/goldforge.py passport <entity_id> --names names.json

# 1–2 · ЖАТВА и ДОСЬЕ — исчерпывающий сбор с провенансом
python3 tools/goldforge/goldforge.py harvest <entity_id>

# 4 · ГЕЙТ ЗОЛОТА — покрытие досье; золото только при исчерпании
python3 tools/goldforge/goldforge.py audit <entity_id> --card card.json
```

`names.json` — выгрузка `entity_names` (все lang: ru / en / iast / alias):

```sql
SELECT value FROM entity_names WHERE entity_id = '<id>';
```

## Что на выходе

- `docs/dossiers/passports/<id>.json` — паспорт имён (**в репозитории**)
- `docs/dossiers/<id>.dossier.json` — досье: пассажи с точной ссылкой
  (файл + смещение + форма). **Артефакт сборки, не коммитится** (велик).

Досье — сырьё мини-книги. Ничего не выдумано: каждый пассаж имеет провенанс
(ЗКН-БТ001). Стадия 3 (сборка) — курируемая: досье → хронология жизни, учение,
труды, наследие; каждое утверждение со ссылкой до стиха (ЗКН-БТ003/БТ006).


## Уровни уверенности (ЗКН-П009)

Жатва **ничего не выбрасывает** — она маркирует:

| уровень | признак | судьба |
|---|---|---|
| **уверенно** | имя + уточнитель рядом («дас», «тхакур») | в сборку |
| **кандидат** | голое имя («просто Нароттама») | решает куратор |
| **омоним** | нарицательное («нароттама» = «лучший из людей», ШБ 1.13.28) | не в сборку |

Жёсткий фильтр запрещён: на Нароттаме он уничтожал **90 из 198** пассажей (45%).

```sh
python3 tools/goldforge/goldforge.py passport <id> --names names.json \
  --qualifier "дас" --qualifier "тхакур" --qualifier "das" --qualifier "thakur" \
  --homonym "дхира" --homonym "лучший из людей"
```
