# Канонические имена и формы — стандарт

Единый стандарт устойчивых имён во всём приложении. Применяется к UI-строкам,
заголовкам, описаниям, лидам и любому тексту, где упоминаются центральные
личности. Внутренние ID и slug-и (`krishna`, `chaitanya`, `prabhupada`,
`krishna-lila`, `gauranga-lila`) — не трогаем, это технические ключи.

## Личности

| Канон | Не использовать (заменяем) |
| --- | --- |
| **Шримати Радхарани** | «Радхарани» в одиночку как имя (в компаундах «Радха-кунда», в составных Sanskrit-формах «игры Радхи и Кришны» — оставляем) |
| **Кришна** | (без изменений) |
| **Гауранга Махапрабху** | «Шри Чайтанья Махапрабху», «Шри Чайтанья», «Чайтанья» — когда речь о Махапрабху в общем, о Навадвипа-лиле, о Панча-таттве |
| **Шри Кришна Чайтанья Махапрабху** | «Шри Чайтанья Махапрабху», «Шри Чайтанья» — когда речь идёт о Гауранге в контексте санньясы (Пури-лила, последние 18 лет в Нилачале) |
| **Шрила Прабхупада** | «Прабхупада» в одиночку. В официальной полной форме: «Его Божественная Милость А.Ч. Бхактиведанта Свами Шрила Прабхупада». «Прабхупада-лиламрита» — название книги, не трогаем. |

## Лилы

| Канон | Не использовать |
| --- | --- |
| **Гауранга Лила** | «Гауранга Лила», «Гауранга-лила», «лила Чайтаньи», «лила Махапрабху» |
| **Кришна Лила** | «Кришна-лила» (как именующая лилу, не slug) |

Оба слова с заглавной во всех падежах:
- Гауранга Лила / Гауранга Лилы / Гауранга Лиле / Гауранга Лилу / Гауранга Лилой
- Кришна Лила / Кришна Лилы / Кришна Лиле / Кришна Лилу / Кришна Лилой

## Когда «Гауранга Махапрабху» vs «Шри Кришна Чайтанья Махапрабху»

Имя «Кришна Чайтанья» Господь принял как санньясу в Катве в 24 года. С этого
момента и до ухода Он жил в Пури (Нилачала) ещё 18 лет. Поэтому:

- **Гауранга Махапрабху** — Навадвипа-лила (рождение, Шачи-Джаганнатха, школа,
  семейная жизнь, Панча-таттва, санкиртана в Маяпуре, Шриваса Тхакур и т.д.).
- **Шри Кришна Чайтанья Махапрабху** — после санньясы: путешествия по Южной
  Индии, жизнь в Пури, ежедневный даршан Джаганнатхи, разлука в настроении
  Шримати Радхарани, последние 18 лет в Гамбхире.

## Исключения

Канон не применяется внутри:
- **названий книг**: «Шри Чайтанья-чаритамрита», «Чайтанья-бхагавата»,
  «Чайтанья-мангала», «Прабхупада-лиламрита» — официальные заголовки.
- **slug-ов и id-шников**: `chaitanya`, `prabhupada`, `gauranga-lila`,
  `krishna-lila` — технические ключи в базе и URL.
- **прямых цитат** из писаний и Прабхупады — никогда не редактируем чужой
  голос.
- **компаундов с дефисом, где «Радха» — часть имени места/практики**:
  Радха-кунда, Радха-Дамодара, Радха-Говинда и т.п.

## Применение

Любая новая UI-строка, longform-секция, описание дхамы/тиртхи/прасада/книги
должна проходить через этот канон. При сомнении — берём более полную форму
(«Гауранга Махапрабху», «Шрила Прабхупада», «Шримати Радхарани»).
