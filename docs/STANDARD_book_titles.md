# Стандарт написания названия книги

Единый источник истины — `apps/web/src/books.ts`.

## Модель

В `BookData` название хранится двумя полями **только ради двухстрочной вёрстки
обложки**:

- `titleLine1` — крупная первая строка;
- `titleLine2` — вторая строка (опционально).

Это **не** «название + подзаголовок». Для книг с дефисным именем `titleLine2` —
это вторая половина одного слова, а не подзаголовок.

## Канон

Канонической формой названия считается **одна строка**, которую возвращает
`bookFullTitle(book)`:

| work | titleLine1       | titleLine2     | Каноническое название        |
|------|------------------|----------------|------------------------------|
| bg   | `Бхагавад-гита`  | `как она есть` | **Бхагавад-гита как она есть** |
| cc   | `Шри Чайтанья-`  | `чаритамрита`  | **Шри Чайтанья-чаритамрита**  |
| sb   | `Шримад-`        | `Бхагаватам`   | **Шримад-Бхагаватам**         |
| brs  | `Нектар`         | `преданности`  | **Нектар преданности**        |

Правило склейки: если `titleLine1` заканчивается дефисом — соединяем без
пробела (часть одного слова), иначе через пробел.

Канон совпадает с названиями в API (`apps/api/src/routes/books.ts`,
`content.ts`). При расхождении правится так, чтобы совпадало с этой таблицей.

## Где что используется

**`bookFullTitle(book)` — везде, где название выводится как одна строка:**

- шапка ридера (sticky header);
- системный/мини-плеер, заголовок аудиосессии;
- нативный шеринг (`navigator.share` → `title`);
- имя PDF-файла при выгрузке;
- QR-карточка книги (главный заголовок);
- контекст формы «Сообщить об ошибке»;
- `alt` обложки.

**`bookShareTitle(book)`** = `«<Канон>. ISKCON ONE LOVE. ИСККОН.»` — для
Open-Graph / share-заголовка (клиент + edge-инъектор OG).

**`titleLine1` + `titleLine2` раздельно** — ТОЛЬКО там, где это сознательная
двухстрочная вёрстка с собственным стилем каждой строки:

- герой-карточка книги (`BookHeroCard`) — учитывает `uniformTitle`;
- титульная страница PDF (`BookDetailPage`).

## Запрещено

- Выводить голый `titleLine1` как самостоятельное название — даёт обрубок
  («Шримад-», «Шри Чайтанья-»).
- Склеивать `titleLine1`/`titleLine2` руками через пробел — даёт «Шримад-
  Бхагаватам». Всегда через `bookFullTitle()`.

## Добавление новой книги

Новая книга = новая запись в `BOOKS` (`books.ts`). Разбить название на
`titleLine1`/`titleLine2` так, чтобы `bookFullTitle()` дал каноническую форму;
для целостного (не «титул+подзаголовок») названия поставить `uniformTitle: true`.
