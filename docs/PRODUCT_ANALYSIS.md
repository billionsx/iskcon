# Аудит gaurangers.com — логика, состояние, дорожная карта

_Глубокий разбор: что строим, что сделано, что синхронизировано с D1, что
работает / заглушка / отсутствует, и переустройство навигации._

## 1. Север (north star)
Ежедневное «купание в голока-према-дхане» для практикующего садху ИСККОН.
Приложение должно быть **садхана-центричным**: каждое утро — даршан, джапа,
чтение, киртан, сверка с календарём. Контент-каталог (книги, личности, дхамы,
центры) — фундамент, но не самоцель.

## 2. Карта приложения сегодня + связь с БД

**A. Библиотека (ядро) — Книги.** 16 книг; ридер (стих/проза/глава);
санскрит/пословный/перевод/комментарий; серверный PDF (Chromium); типографика
BBT (`Skt`); поиск по каталогу (`bookSearch.ts` — фаззи, IAST, транслит).
D1: тексты (`works/divisions/verses/verse_texts/verse_tokens`). Прогресс чтения —
ДВА канала: сервер `/me/progress` (Кабинет) + локальный `reading.ts` (полка
«Продолжить», офлайн/гость). Закладки, история — сервер.

**B. Аудио — Киртаны/Бхаджаны.** Артисты, треклисты (Internet Archive), плеер
(Media Session, мини-плеер). D1: бхаджаны (`/bhajans`); прослушивания
(`/me/listen`); треки — IA-идентификаторы (статика).

**C. Садхана (ежедневная практика) — спрятана в подтаб Главной.**
Живое: Джапа (D1 `japa_round`), Дневник (D1 `sadhana_day` + минуты чтения),
Даршан дня (D1 `darshan`, живой источник из храмов), Обет/санкальпа (`vows.ts`
— ТОЛЬКО localStorage), Прасад (100 рецептов — статический `prasad.ts`).
Заглушки («Скоро», без обработчика): Стих дня, Экадаши/посты, Новости ИСККОН,
«Мой прогресс» (экран), Закладки/заметки (экран), Путь преданного.

**D. ИСККОН-справочник — раздут как «Главная» (10 подтабов).**
Календарь (D1 `gcal_days`, экадаши/явления), Лента (@iskcone), Центры/Ятра
(полный CRUD D1 + публичный локатор `/centers` + карта + кабинет `/my/centers`
+ модерация), Рестораны (из центров), Образование/Медиа (внешние ссылки),
Структура/Документы/Ссылки (статика), Личности/Герои (D1 `/personalities`),
Дхамы (статический `dhamas.ts` + карта).

Служебное: Кабинет (auth D1 `user_auth/sessions`; профиль; статистика;
«Продолжить»; история; закладки; заметки гибрид; садхана-карта). Магазин
(Apple Pay/карта/СБП/USDT; заказ → D1 + email; `cart.ts` localStorage). Донат.

## 3. Работает / полу-готово / нет
- ✅ **Работает:** чтение+PDF+поиск книг, плеер, джапа, дневник, даршан,
  календарь, центры (CRUD+локатор+карта), прасад-рецепты, кабинет+auth,
  корзина+заказ, заметки, обет.
- 🟡 **Риск:** прогресс чтения двухканальный (локальный↔серверный расходятся
  после логина/смены устройства); обет и корзина — только localStorage
  (теряются при смене устройства/выходе); прасад и дхамы — статика (правка
  только релизом).
- 🔴 **Нет/заглушка:** глобальный поиск (движок+иконка готовы, экрана нет);
  стих дня; экадаши-трекер; путь преданного; новости ИСККОН на русском;
  экраны «мой прогресс»/«закладки» из хаба садханы.

## 4. Разрывы до «лучшего в мире для садху»
1. **Садхана закопана** — джапа/даршан/чтение/экадаши на 3-м уровне; должны
   быть на 1-м касании.
2. **Прайм-слот занят коммерцией** — корзина в верх-лево; первое касание не
   должно быть про магазин.
3. **Нет глобального поиска** — при готовом движке/иконке/маршруте.
4. **«Главная» — свалка** (практика + справочник + документы вперемешку).
5. **Нет ежедневных крючков возврата** — стих дня, экадаши-напоминание,
   серия садханы на виду, пуши.

## 5. Переустройство навигации

### Верхний бар
`[Корзина] — [логотип] — [♥]` → **`[🔍 Поиск] — [логотип] — [♥ Избранное]`**.
Поиск глобальный (книги, стихи, киртаны, бхаджаны, рецепты, центры, личности,
дхамы) — `bookSearch.ts` расширяется на все индексы; `AISearchIcon` уже в коде;
`/search` зарезервирован → это «дошить». Корзина → контекстно (на странице
книги/товара) + в Кабинете.

### Нижние табы (≤5, Apple HIG) — по осям частота × объём для садху

| Раздел | Частота | Объём | Вердикт |
|---|---|---|---|
| Садхана (джапа/даршан/дневник/экадаши/стих) | ежедневно | растёт | **таб #1 (дом)** |
| Книги | ежедневно | огромный | **таб #2** |
| Аудио (киртаны+бхаджаны+лекции) | ежедневно | большой | **таб #3** |
| Календарь | ежедневно (сверка) | средний | → ИСККОН-хаб |
| Центры/Рестораны/Дхамы/Герои/Образование/Структура/Документы/Лента | эпизодически | большой суммарно | **таб #4 (ИСККОН-хаб)** |
| Кабинет | служебно | — | **таб #5** |

**Раскладка:** 1) **Садхана** (старт) — даршан, стих, джапа, дневник+серия,
экадаши. 2) **Книги**. 3) **Слушать** — киртаны/бхаджаны/лекции. 4) **ИСККОН**
— нынешняя «Главная» как справочник-хаб (календарь, лента, центры, рестораны,
дхамы, герои, образование, структура, документы). 5) **Кабинет** (+ корзина).

## 6. Дорожная карта
- **Tier 0 (структура):** поиск в прайм-слот + глобальный поисковый экран;
  переустройство таб-бара (Садхана-дом / Книги / Слушать / ИСККОН / Кабинет).
- **Tier 1 (ежедневные крючки):** стих дня; экадаши-трекер (данные в
  `gcal_days` уже есть); серия садханы на виду; синхронизация обета на сервер.
- **Tier 2 (глубина):** путь преданного; экраны «прогресс»/«закладки»;
  новости на русском; реконсиляция двухканального прогресса; прасад/дхамы → D1;
  автоверификация платежей (ЮKassa/TronGrid).
