# ISKCON DESIGN

> Дизайн-система приложения **ISKCON ONE LOVE**. Источник истины домена **Д**
> свода `docs/LAWS.md`. Утверждена 11.07.2026.
>
> Единственный источник цвета, шрифта, отступа, радиуса и движения.
> Магические числа в компонентах запрещены (ЗКН-Д001).

---

## 1. Слои системы

```
ui/globals.css      CSS-переменные — база, тема (светлая/тёмная) применяется сама
      ↓
ui/tokens.ts        TS-зеркало: tk.text.footnote вместо магического 13
      ↓
ui/primitives.tsx   кирпичи: Card · SectionHeader · Chip · Stat
      ↓
компоненты          собираются ТОЛЬКО из кирпичей и токенов
```

Правка значения — **в одном месте** (globals.css), видна везде.

## 2. Цвет — фирменное золото

| Токен | Значение | Назначение |
|---|---|---|
| `--color-gold` | `#D2AA1B` | Фирменное золото: логотип ИСККОН, надписи-классификации ВБК/ВСК, акцент цитат |
| `--color-gold-deep` | `#9c7c15` | Глубокий акцент интерфейса: ссылки, активные состояния |
| `--color-gold-50…900` | рампа из `#D2AA1B` | Шкала оттенков |

Золото логотипа и золото кода — **один и тот же** `#D2AA1B` (проверено по SVG).

**Отменено:** `--color-brand-blue` — токен назывался «синим», а хранил золото;
шкала наполовину оставалась синей (мёртвый код шаблона). Вычищено (ЗКН-Д003).

## 3. Шрифт

| Токен | Гарнитура | Где |
|---|---|---|
| `--font-display` | дисплейная | Имена, заголовки |
| `--font-text` | текстовая | Весь UI |
| `--font-scripture` | **Gentium Book Plus** | Стихи, транслитерация, термины, цитаты (ЗКН-С001/С002) |

`--font-scripture` — единственный шрифт с корректными диакритиками над кириллицей
(Georgia их «роняет»). Курсивом им набираются термины и цитаты — «чужой голос».

---

## 4. Планка (ЗКН-Д004)

Уровень **Apple / iOS 2026**: роли Dynamic Type, сетка 4pt, стеклянные поверхности,
предсказуемое поведение. Эстетика — тихая, минималистичная (не «Liquid Glass»).
Колонка контента — **480px** по центру на десктопе.

---

## 5. Заглушка обложки (ЗКН-Д005)

Нет загруженного изображения → **золотой логотип ИСККОН на белом фоне**.
Единый ассет `apps/web/public/cover-fallback.svg` для всех типов: личности, книги,
аудио, бхаджаны, места, рецепты. Локальные заглушки (буква-инициал, `audio-cover*.png`)
отменены.

## Повторяющийся блок — из примитивов (ЗКН-Д002)

Общие функции и строки списков живут в `ui/primitives.tsx`, а не копируются по
файлам. Живой пример: русское число `plural(n, "книга", "книги", "книг")` лежало
слово-в-слово в СЕМИ файлах (`BooksHub`, `KirtansScreen`, `DhamaScreen`,
`DhamaDetailPage` и под псевдонимами `pluralRu`, `pluralDays`). Семь копий —
семь мест, куда правка не доедет.

Гейт: `laws-lint.py::check_d002_primitives` — объявление `plural*` вне
`ui/primitives.tsx` валит линтер.
