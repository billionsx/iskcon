# docs/sources — первоисточники для ПКЛ

Оригинальные документы-источники для набора информации в карточки личностей.
BBT-точность, ноль фабрикации; при обогащении карточки — точная атрибуция и ссылка на стих.

## Опись

- `Narahari_Cakravarti_Thakura_Bhaktiratnakara.pdf` — «Бхакти-ратнакара» Шрилы Нарахари Чакраварти Тхакура, ПОЛНЫЙ текст (374 стр., англ. перевод). Основной источник по лилам Госвами Вриндавана и тиртхам Враджа.
- `Bhakti_Ratnakara_Excerpts.pdf` — те же «Бхакти-ратнакара», краткие выписки (6 стр., перевод Advaitadas 1986–1996): лилы Санатаны, Рупы, Рагхунатхи даса Госвами; тиртхи; визит Джахнавы на Радха-кунду. Удобно для быстрой навигации.
- `Narahari_Cakravarti_Thakura_Songs.pdf` — песни Нарахари Чакраварти Тхакура (ISKCON Media Vedic Library).
