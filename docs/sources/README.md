# docs/sources — Библиотека первоисточников для карточек личностей (ПКЛ)

Оригинальные книги и документы, из которых берётся информация для карточек
личностей библиотеки ISKCON ONE LOVE. Растущий ресурс — пополняется по мере
надобности.

Требование неизменно: **BBT-точность, ноль фабрикации.** При обогащении карточки —
точная атрибуция «автор слов → личность» и ссылка на конкретный стих (см.
`docs/pkl-hero-playbook.md`, Фаза 0 «Паспорт источника»).

---

## 1. Как этим пользоваться (для будущих сессий)

1. **Ищи по `.txt`, а не по PDF.** Рядом с каждым годным источником лежит извлечённый
   плоский текст `*.txt` — по нему работает `grep`/`instr`. PDF/DOC — это исходники-архивы.
   ```
   grep -rn "имя-стем" docs/sources/            # по всему корпусу
   grep -rn "Санатан" docs/sources/hagiography/ # по разделу
   ```
2. **Учитывай склонения и мирские имена.** Кириллица — искать по основе слова (`Санатан`,
   а не `Санатана`); санскрит в англ. текстах — по стему без диакритики (`Sanatan`,
   `Raghunath`). До-санньясные / мирские имена см. в паспорте источника ниже.
3. **`.txt` из PDF содержат постраничные метки `[p.N]`** — они соответствуют номеру
   страницы PDF и годятся для точной ссылки. `.txt` из `.doc` меток страниц не имеют
   (Word перетекает) — ориентир в них: заголовки глав/волн («CHAPTER 7», «Пятая Волна»).
4. **Первоисточник vs вторичный.** Стихи и комментарии цитируются ТОЛЬКО из
   первоисточников (помечены `ПЕРВОИСТОЧНИК`). Современные компиляции (`ВТОРИЧНЫЙ`:
   Роузен, Капур, «Lives of the Saints») — для фактов, ориентировки и поиска эпизодов,
   но НЕ как источник дословных цитат-шастр.

## 2. Технические особенности (важно)

- **PDF от ISKCON Media Vedic Library — кастомный PUA-шрифт.** У них «текстовый слой»
  есть, но все глифы отображены в приватную область Unicode (U+F000+). `pdftotext`
  выдаёт мусор. Реальный человекочитаемый текст этих трудов взят из парных `.doc`
  (LibreOffice → чистый UTF-8) и лежит в `*.txt`. Затронуты: Бхакти-ратнакара,
  Нароттама-виласа, Матхура-мандала, Songs, Кришна-Бхаджанамрита.
- **OCR-файлы (`*.ocr.txt`) — черновики.** Где нет `.doc` и PDF в PUA-шрифте
  (Кришна-Бхаджанамрита), текст получен tesseract'ом. Точность высокая, но сверять
  диакритику/санскрит по PDF-странице.
- **Диакритика IAST в англ. первоисточниках** (Chaitanya-charita, excerpts) извлекается
  в лигатурной раскладке BBT-шрифта (`ä å ç ñ é ù …`). Это нормально и ищется по
  латинской основе; при вставке в карточку — приводить к обычному тексту, курсив
  накладывает глоссарий рендера (см. `src/ui/scripture.ts`).
- **`_variants/`** в папке труда — неполные/дублирующие редакции (обрывочные `.doc`,
  «incomplete» PDF, альт. выписки). Держим для полноты, в работе не нужны.

  chaitanya-lila/                   Канонные биографии Махапрабху (ЧБ, ЧМ, Маха-кавья)
  prabhupada-shikshamrita/          Шикшамрита — наставления Прабхупады (RU)

распознанный; `★` ниже помечает **годный для поиска файл** каждого труда.

## 4. Каталог

### 4.1 Бхакти-ратнакара · `bhakti-ratnakara/`
**Автор / NARRATOR:** Нарахари Чакраварти Тхакур (он же Гханашьяма дас). `ПЕРВОИСТОЧНИК`.
15 волн (глав), ~15 019 стихов. Главный биографический свод по эпохе после Махапрабху:
Шесть Госвами Вриндавана, Шринивас Ачарья, Нароттама дас, Шьямананда, визит Джахнавы
на Радха-кунду; тиртхи Враджа и Навадвипы.
- ★ `Bhakti-Ratnakara.EN.txt` — полный англ. текст (из `.doc`, чистый). Якоря: `CHAPTER N`.
- `Bhakti-Ratnakara.EN.pdf` — полный PDF (374 стр., PUA-шрифт → только для человека).
- `Bhakti-Ratnakara.EN.doc` — исходник чистого текста.
- ★ `Bhakti-Ratnakara.excerpts.EN.txt` / `.excerpts.EN.pdf` — краткие выписки (перевод
  Advaitadas): лилы Санатаны, Рупы, Рагхунатхи даса; тиртхи; Джахнава на Радха-кунде.
  Удобно для быстрой навигации по эпизодам.
- `_variants/` — «incomplete» PDF (278 стр.), альт. выписки, обрывочные `.doc`.

### 4.2 Нароттама-виласа · `narottama-vilasa/`
**Автор / NARRATOR:** Нарахари Чакраварти Тхакур; перевод Dravida das. `ПЕРВОИСТОЧНИК`.
12 глав — продолжение БР, жизнь Нароттамы даса Тхакура; также Шринивас, Шьямананда,
фестиваль Гаура-пурнимы в Кхетури.
- ★ `Narottama-Vilasa.EN.txt` — с метками `[p.N]` (из текстового PDF).
- `Narottama-Vilasa.EN.pdf` (66 стр., реальный текст) · `Narottama-Vilasa.EN.doc`.
- `_variants/` — PDF от ISKCON Media (PUA), обрывочные `.doc`.

### 4.3 Матхура-мандала Парикрама · `mathura-mandala-parikrama/`
**Источник:** 5-я Волна «Бхакти-ратнакары», Нарахари Чакраварти. `ПЕРВОИСТОЧНИК`.
Парикрама по Врадже/Матхура-мандале, тиртхи и связанные лилы — опора для гео-библиотеки
дхамы и карточек Госвами.
- ★ `Mathura-Mandala-Parikrama.RU.txt` — русский перевод 5-й Волны (из `.doc`, чистый).
- `Mathura-Mandala-Parikrama.RU.doc` — исходник.
- `Mathura-Mandala-Parikrama.EN.pdf` — англ. PDF (82 стр.) в PUA-шрифте: **машинного
  текста нет, при нужде англ. — OCR.** Русский `.txt` покрывает содержание.

### 4.4 Песни Нарахари Чакраварти · `narahari-songs/`
**Автор:** Нарахари Чакраварти Тхакур. `ПЕРВОИСТОЧНИК` (бхаджаны, транслитерация+перевод).
- ★ `Narahari-Songs.EN.txt` (из `.doc`) · `Narahari-Songs.EN.doc` · `Narahari-Songs.EN.pdf` (PUA).

### 4.5 Кришна-Бхаджанамрита · `krishna-bhajanamrta/`
**Автор:** Нарахари Саркар Тхакур (спутник Панча-таттвы — **не** Нарахари Чакраварти!).
`ПЕРВОИСТОЧНИК`. Трактат в стихах о вайшнавском этикете и почтении к преданным.
- ★ `Krishna-Bhajanamrta.EN.ocr.txt` — OCR (11 стр.), сверять по PDF.
- `Krishna-Bhajanamrta.EN.pdf` — PUA-шрифт.

### 4.6 Шикшамрита (наставления Прабхупады) · `prabhupada-shikshamrita/`
**Автор слов:** А.Ч. Бхактиведанта Свами Прабхупада (`by = Шрила Прабхупада`).
`ПЕРВОИСТОЧНИК` для его собственных слов. Свод писем/наставлений по темам и датам
(формат «68-09» = год-номер). 1000 стр., русский.
- ★ `Prabhupada-Shikshamrita.RU.txt` — метки `[p.N]`.
- `Prabhupada-Shikshamrita.RU.pdf`.

### 4.7 Биографические своды · `hagiography/`  (`ВТОРИЧНЫЙ` — факты, не цитаты-шастры)
- **«Lives of the Saints» (EN, 408 стр.)** — ★ `Lives-of-the-Saints.EN.txt`.
  Широкий свод ~37 гаудия-святых, по главе на каждого (см. индекс §5).
- **О.Б.Л. Капур (Адикешава дас), «Святые Враджа» (RU, 358 стр.)** —
  ★ `Kapoor_Saints-of-Vraja.RU.txt`. Ок. 45 очерков о **поздних сиддхах-бабаджи Враджа
  XVIII–XX вв.** (Джаганнатха дас Бабаджи, Гауракишор Широмани, Лала Бабу,
  Кришнапрем/Р. Никсон и др.) — уникальный пласт, НЕ классическая эпоха.
- **Сатьярадж дас (С. Роузен), «Вайшнавские святые» (RU)** —
  ★ `Rosen_Vaishnava-Saints.RU.txt`. Посвящена **Шринивасу Ачарье, Нароттаме дасу,
  Шьямананде**; введение — ценный обзор бенгальских первоисточников.
- **С. Роузен, «Шесть Госвами Вриндавана» (RU, 88 стр.)** —
  ★ `Rosen_Six-Goswamis-of-Vrindavan.RU.txt`. Рупа, Санатана, Рагхунатха дас,
  Рагхунатха Бхатта, Джива, Гопала Бхатта.

### 4.8 Канонные биографии Гауранга-лилы · `chaitanya-lila/`
Три канонных жизнеописания Гауранга Махапрабху в одном разделе. Все — `ПЕРВОИСТОЧНИК`;
основной пласт для карточек Самого Господа и Его спутников Навадвипы (Панча-таттва,
семья, ранние преданные).
- **Чайтанья-Бхагавата** — Вриндаван дас Тхакур, с комментариями Шрилы Бхактисиддханты
  Сарасвати Тхакура. Полная книга: три кханды (Ади, Мадхья, Антья) — от явления Господа
  до Его лет в Нилачале, преимущественно Навадвипа-лила. В конце — большой **«Словарь
  имён и терминов»** (сотни статей: удобный справочник по спутникам и понятиям).
  - ★ `VrindavanaDasa_Chaitanya-Bhagavata.RU.txt` — полный рус. текст (из `.doc`, чистый).
    Якоря: `Глава N`, «Ади-/Мадхья-/Антья-кханда», колофоны «Так заканчивается N глава …
    кханды». Меток `[p.N]` нет.
  - `VrindavanaDasa_Chaitanya-Bhagavata.RU.doc` — исходник (3603 стр.; цифровое изд.
    saraswati.pro по печатному Шри Чайтанья Гаудия Матха 1987).
- **Чайтанья-мангала** — Лочана дас Тхакур; перевод Субхаг Свами, ред. Маханидхи Свами.
  Ранняя бенгальская классика XVI в., четыре кханды (Сутра, Ади, Мадхья, Шеша). Уникальное
  содержание: нисхождение Радхи-Кришны как Гауранги, беседы Нарады, юга-аватары, лила
  Навадвипы и принятие санньясы.
  - ★ `LochanaDasa_Chaitanya-Mangala.RU.txt` — полный рус. текст (из `.doc`). Якоря:
    «Сутра-/Ади-/Мадхйа-/Шеша-Кханда». *В нескольких строках — вкрапления битой cp1251
    (артефакт исходного `.doc`); на поиск по именам почти не влияет.*
- **Шри Кришна-Чайтанья-чарита Маха-кавья** — Кави Карнапура (Парамананда Сен), EN+санскрит,
  741 стр. Санскритская кавья с ПОСЛОВНЫМ переводом; ссылки на уровне стиха (`Prakrama`/
  `Sarga`), IAST в BBT-шрифте.
  - ★ `KaviKarnapura_Chaitanya-Charita-Maha-Kavya.EN.txt` (метки `[p.N]`) · `.EN.pdf`.

### 4.9 Дхама-махатмья · `dhama-lila/`
- **Бхактивинода Тхакур, «Шри Навадвипа-дхама-махатмья» (Парикрама-кханда, RU, 352 стр.)**
  — ★ `Bhaktivinoda_Navadvipa-Dhama-Mahatmya.txt`. `ПЕРВОИСТОЧНИК`. Девять островов
  Навадвипы, лила Гауранги Махапрабху в Навадвипе, слава дхамы — опора Гауранга-лилы и гео-дхамы.

## 8. Как пополнять папку

1. Клади новый файл в тематическую подпапку (или заведи новую по труду/теме).
2. Имя: `Название[.Автор].LANG.ext`.
3. Если PDF от ISKCON Media или иной PUA-шрифт — сразу проверь `pdftotext` на реальный
   текст; при мусоре ищи `.doc`-версию или делай OCR, клади результат в `*.txt` (`*.ocr.txt`).
4. Из текстовых PDF извлекай `.txt` с метками `[p.N]`.
5. Впиши труд в §4, §5 (индекс личностей) и §6 (паспорт источника).
