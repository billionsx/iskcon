## §0. Три столпа, отличающие оригинал от подделки

1. **МАТЕРИАЛ.** У Apple ни одна плашка не «покрашена» — она сделана из стекла:
   размытие фона + насыщение + верхний блик кромки + нижняя тень кромки +
   волосяная обводка. Плоская заливка `rgba` без этих четырёх слоёв — подделка.
2. **ДВИЖЕНИЕ.** Ни одно состояние не сменяется скачком: всё приезжает пружиной,
   всё нажатое отвечает в первые 100 мс, всё исчезающее успевает уйти.
   Мгновенная подмена DOM — подделка.
3. **РОЛИ, А НЕ ЧИСЛА.** Кегль, трекинг, интерлиньяж живут парой «роль → тройка
   значений» (§3). Полукегли и вес 800 в тексте запрещены.

📐 IMG_2336: заливка вкладок над чёрным даёт ≈ `#1A1A1A`; над контентом плашка
светлеет — значит заливка **полупрозрачно-серая, не тёмная**.
```css

-webkit-backdrop-filter: blur(34px) saturate(1.85);
backdrop-filter: blur(34px) saturate(1.85);
box-shadow:
  inset 0 1px 0 rgba(255,255,255,.16),   /* верхний блик линзы */
  inset 0 -1px 0 rgba(0,0,0,.22),        /* нижняя тень кромки */
```

### 1.3 Стекло поверх артворка (кнопки и капсулы внутри плеера)

## §5. ДВИЖЕНИЕ — закон анимации

Все длительности с 🎞 сняты покадрово с записи 60 fps (профиль межкадровой
энергии); кривые — пружины iOS, перегиб без отскока у листов и с лёгким
перелётом у поповеров.

### 5.0 Кривые (токены)
```css
--ease-out:  cubic-bezier(.25, 1, .35, 1);     /* разгон-выбег контента   */
--spring:    cubic-bezier(.34, 1.30, .40, 1);  /* поповеры, отпускание    */
--sheet:     cubic-bezier(.32, .72, 0, 1);     /* листы: плеер, шторки    */
```

### 5.1 Нажатие (везде)
Вход ≤ 120 мс, линейно; отпускание 300–350 мс `--spring`.
- Хром-стекло и круги: `scale(.96)` + светлеющая заливка.
- Транспорт плеера: `scale(.86)`.
- Ряды/строки/пункты меню: заливка-вспышка `rgba(255,255,255,.07)` мгновенно,
  гаснет 350 мс.
- Карточки полок: `scale(.97)`.

### 5.2 Плеер: открытие/закрытие — МОРФ пилюли, не шторка
🎞 open_013–047 (60 fps): плеер НЕ въезжает листом. **Мини-пилюля сама
раздувается в карточку плеера**: прямоугольник пилюли → весь кадр, обложка
летит из гнезда дока (36, R7) в большую (345, R12), содержимое проявляется на
второй половине пути. Снятая кривая (доля пути): ~28 % к 45 мс, ~56 % к 90 мс,
~81 % к 165 мс, дальше хвост ×0.78/кадр до ~560 мс. Перелёта нет.
Закон: FLIP через WAAPI, длительность **560 мс**, кривая
`linear(0, .28 8%, .56 17%, .66 20%, .81 30%, .9 36%, .94 42%, .965 50%,
.978 57%, .988 66%, .994 78%, .998 89%, 1)`; карточка растёт трансформом из
прямоугольника пилюли (радиус компенсируется 24/s), призрак обложки летит
отдельно вне масштаба, настоящая обложка на время полёта спрятана.
Закрытие: обратный полёт **400 мс `cubic-bezier(.32,.72,0,1)`** (🎞 108.55–
108.93 с ≈ 383 мс), содержимое гаснет к 35 % пути. Свайп вниз: лист следует
пальцу; отпускание >130 pt — обратный морф из текущего положения.
Фон при морфе дымится нарастающе (🎞 open_016–018: ~.1→.22 к 100 мс) — нижний
слой сжимается до `scale(.94)` с `brightness(.6)`.

### 5.3 Плеер: смена вида (обложка ↔ лирика ↔ очередь)
🎞 54.5 с и 71.5 с: смена быстрая, ~200–320 мс, обложка не исчезает, а
**сжимается к левому верхнему углу** в миниатюру шапки (и обратно растёт).
Закон: исходящий вид 220 мс `--ease-out` (обложка — scale к origin 0 0 до
миниатюры, гаснет на последней трети); входящий 260–320 мс (обложка растёт из
угла; шапка приподнимается на 8 pt и проявляется). Нижний транспорт неподвижен.

### 5.4 Меню
🎞 b/mn_004–009: короб у якоря растёт 43→82 pt за ~5 кадров — рост ИЗ ТОЧКИ,
не проявление почти-готового. Кривая целиком кадрово не снята (подложка глушит
дифф) — честное приближение: вход 380 мс `--ease-out`, scale .18→1.035 (62 %)
→1, `transform-origin` в якоре; затемнение+блюр подложки 280 мс.
**Выход обязателен**: 160 мс ease, scale→.85, прозрачность→0; подложка гаснет
вместе. Мгновенное исчезновение запрещено.

### 5.5 Шторки
Вход 420 мс `--sheet` снизу; выход 300 мс `--sheet` вниз; затемнение синхронно.

### 5.6 Пауза дышит обложкой
🍎 фирменное поведение плеера: на паузе большая обложка садится к
`scale(.785)` (тень мельче), при пуске встаёт в 1.0. Пружина 500 мс `--spring`.

### 5.7 Полоса времени и громкость
Покой: высота 7, заливка `rgba(255,255,255,.28)` / ход `.78`.
Палец на дорожке: высота **11**, дорожка `.36`, ход `#FFF`, времена белеют —
220 мс; отпускание обратно 300 мс. Ход ползёт `linear` между тактами.

### 5.8 Смена вкладки
🎞 116.5 с: контент — быстрый кроссфейд ~100–150 мс (без слайда);
знак активной вкладки отвечает лёгким «поп» 300 мс `--spring`.

### 5.9 Переход push/pop
Вперёд: экран входит сдвигом +56 pt → 0 с проявлением, 360 мс `--ease-out`.
Назад: вход с −40 pt → 0, 300 мс. (⚠ у Apple уходящий экран ещё и параллаксит
под входящим; однослойная реализация фиксирует главное — направление и выбег.)

### 5.10 Морф дока (Liquid Glass)
📐 dock_static_mask: полный — мини 351×48 врезка 21, вкладки 62 (8.7+44.6+8.7)
со знаками и подписями, круг 62. Компакт — мини **334×43** врезка **29.7**,
капсула **w 264** только знаки (подписи гаснут), высота 57, круг **52.3**,
зазор капсула→круг **15**. Морф ~20 кадров = **350 мс `--ease-out`**, ЕДИНЫЙ
DOM: свёртка — превращение, не замена слоёв. Триггер — НАПРАВЛЕНИЕ прокрутки
(вниз — компакт, вверх — полный, гистерезис ~14 px), у кромки верха всегда
полный. Порог-по-смещению запрещён.

### 5.11 Бегущая строка мини-плеера
Название длиннее капсулы — плывёт: пауза 1.2 с → ровный ход ~28 pt/с →
разрыв 40 pt → повтор; края текстовой зоны растворяются маской 12 pt.

### 5.12 Караоке-строки
Активная строка: чёткая, `scale(1.015)` от левого края; прошлые/будущие в
лёгком блюре; смена 400 мс. Прокрутка — плавная, активная строка у центра.
Точки-ожидание пульсируют каскадом 1.4 с.

### 5.13 Фон плеера живой
Поверх сумрака §2 — два широких радиальных пятна тёплого света дрейфуют
циклами 26 с и 34 с (сдвиги ≤ 6 %, прозрачность ≤ .35): фон дышит, не мигая.
⚠ параметры дрейфа подобраны по характеру записи, числа не измеримы.

### 5.14 Уважение к настройкам
`prefers-reduced-motion: reduce` — все декоративные циклы (§5.11–5.13)
останавливаются, переходы падают до 1 мс.

## §8. Запреты (симптомы «китайской подделки»)

1. Плоская заливка вместо стекла §1.
2. Мгновенная подмена DOM вместо движения §5 (в т.ч. закрытие меню без выхода).
3. Мёртвая кнопка: элемент, который никак не отвечает на нажатие.
4. Полукегли; вес 800 в тексте; трекинг за потолком ±0.4.
5. Число «на глаз» без знака источника (📐/🎞/🍎/⚠/🎨) — правдоподобное число
   хуже отсутствующего (ЗКН-БТ001).
6. Сетка там, где детей меньше колонок (ловушка №2) — раскладку держит flex.
7. Тень на чёрном холсте вне плеера.

## §9. Замеры видео (протокол)

Запись 1180×2556, 60 fps, 197.2 с. Энергия движения = средний |Δ| яркости
между кадрами на 2 fps → окна активности; окна пересняты 60 fps в 393×852.
- 52.2–53.6 с — раскрытие плеера (быстрая фаза ≤ 350 мс + осадка).
- 54.5 с / 71.5 с — обложка ↔ лирика (~200–320 мс, морф к углу).
- 75.5 с — очередь (~220 мс).
- 108.55–108.93 с — закрытие плеера: **383 мс** (профиль ровного выбега, без
  отскока — кривая класса `--sheet`).
- 116.5 с — смена вкладки: всплеск 2 кадров ≈ 100 мс + осадка.

## §11. Метод замера кадров

`pt = ширина/393`. Кегль из чернил (доли 0.714/0.750/0.950) с перекрёстной
проверкой по ширине строки. Границы фигур — по яркости. Радиус — по профилю
края двумя точками. Кромка в цвет фона — сравнением яркости внутри/снаружи.
Проверка минимум на двух кадрах независимо.
