# Apple Books — роспись 46 кадров

Одиннадцатый продукт. До 20.07.2026 Books был **дырой в эталоне** вместе с Фото
и лентой (§12): экраны книг строились без образца, хотя «Книги, священные
тексты» — центральное соответствие всего мульти-приложения.

**Шрифт не берём.** Books набран отдельной гарнитурой; у нас единый SF. Отсюда
берётся **логика**: какие экраны есть, из чего сложены, что с чем связано, как
устроен каталог.

**Как снято.** Геометрия — `tools/ios26-census.py` по кадрам @3x 1179×2556.
Семантика — OCR (tesseract, только `eng`), поэтому кириллические названия книг
в распознавании плывут; интерфейсный хром читается чисто. Названия экранов
опознаны по хрому и структуре, **не глазами**: инструмент `view` картинку не
отдаёт. Числа от этого не страдают — они сняты машиной, — но роспись держится
на OCR, и это помечено как в §12 у четырёх последних продуктов.

## Четыре вкладки

📐 Нижний слой присутствует: **Home · Library · Book Store · Search** (f15, f25,
f43, f46). Поиск — **отдельная вкладка**, а не поле в шапке.

### 1. Home — не каталог, а панель чтения

Это **логика Fitness, приложенная к чтению**, и она сильнее всего относится к нам.

| Блок | Кадр | Содержание |
|---|---|---|
| «Previous» | f30 · f46 | продолжить чтение, **процент прочитанного** прямо на плитке (`PDF · 51%`) |
| «Today's Reading» | f42 | `0:46` из **дневной цели 5 минут** — кольцо |
| Серия | f42 · f45 | «Start a new streak» · «Longest Reading Streak — 3 Days» + **SHARE** |
| «Books Read This Year» | f37 | «0 books finished. Read 3 more books to reach your goal» + **SHARE** · **ADJUST GOAL** |
| «Reading Goals» | f30 | пустое состояние: «Find a great book, set a goal and make reading a daily…» |
| «Explore the Book Store» | f42 | выход в магазин из панели |

**Вывод для нас.** Домашний экран книг у Apple — это не полка, а **садхана
чтения**: дневная цель, серия, годовая цель, доля прочитанного, поделиться.
У нас уже есть Практика с кольцами и обетами — значит книги и садхана
соединяются не «ссылкой», а общей моделью цели и серии.

### 2. Library — коллекции со счётчиками

| Элемент | Кадр | Замечание |
|---|---|---|
| Список коллекций | f03 | Want to Read · Finished **3** · Books **3** · Audiobooks · PDFs **12** · My Samples **1** · Downloaded **1** |
| Счётчик у каждой | f03 | число справа — **сколько внутри**, а не украшение |
| Режим правки | f28 | ручка перетаскивания + **глаз** у каждой строки: порядок и **видимость** коллекции |
| Вид и отбор | f31 | «Select» · «Grid» — список ↔ сетка |
| Внутри коллекции | f20 · f29 · f25 | обложка · название · автор · бейдж **SAMPLE** · **дата добавления** («18 Nov») · «1 book» |
| Действия над книгой | f10 · f32 | Share · Sample · Want to Read · Add to Collection · Mark as Finished · **Suggest More / Suggest Less** |
| Разрушающее | f33 | «Remove from My Samples» · «Remove Download» — **раздельно**: убрать из коллекции ≠ стереть файл |

**Suggest More / Suggest Less** — управление рекомендациями **прямо из карточки
книги**. Человек правит выдачу, не уходя в настройки.

**Разделение «убрать из списка» и «снять загрузку»** — прямо про нашу офлайн-модель.

### 3. Book Store — три уровня каталога

```
Разделы (f27, f36)  →  Категория (f44)      →  Подкатегория (f40)
Fiction & Literature    General Fiction >       полки-карусели
Non-Fiction             Poetry >
FEATURED COLLECTION
Biography
```

| Элемент | Кадр |
|---|---|
| Чарты: «Top Free», нумерация **1 · 2 · 3**, кнопка **GET**, рейтинг ★ | f34 · f35 · f39 |
| Фильтр чарта («Free») | f39 |
| Карточка книги: «From the Publisher» со свёрткой **…More** | f06 · f07 |
| Отзывы: звёзды, «out of 5», **959 Ratings**, «Reviews · Sort» | f08 |
| «More by author» — полка работ автора | f13 |
| «Customers Also Bought» — вторая полка | f16 |

**Нумерованный чарт** — то, чего у нас нет вовсе, а для библиотеки писаний он
естественен: чаще всего читаемое, самое цитируемое.

### 4. Search — история как первый экран

| Элемент | Кадр |
|---|---|
| Пустое состояние с областью поиска («Apple Books ⌄») | f41 |
| **Recently Searched** + «Clear» | f01 · f38 |
| **Recently Viewed** + «Clear» — карточки с обложкой и **GET** | f01 · f38 |
| Подтверждение очистки — алерт с объяснением области действия | f01 |

Алерт не спрашивает «уверены?», а **объясняет последствие**: очистка убирает
историю «в приложении Книги **только на этом устройстве**».

### 5. Читалка

| Элемент | Кадр |
|---|---|
| Баннер «Sample» сверху с × и названием | f17 · f18 · f19 · f22 |
| Счётчик страниц «1 of 17» | f04 |
| **Line Guide** — линейка-указка: «Tap a line above or below the guide, or long-press to drag it» | f04 |
| «Turn Off Line Guide» | f02 |
| **Background Dimming**: High · Medium · Low · None | f02 |

**Line Guide** — доступность чтения, которой у нас нет. Для длинных санскритских
текстов с транслитерацией она даёт ровно то, что нужно: удержать строку.

### 6. Уведомление говорит, КУДА положило

f11: «Added — This book was added to your **Want to Read** list in **Home**, and
a sample was downloaded.» · f12: «Removed — This book was removed from Want to Read.»

Не «Готово», а **адрес результата**. У нас тосты этого не делают.

### 7. Лист «Поделиться»

f21: AirDrop · Messages · Mail · **Notes** · Copy · **Add to New Quick Note** —
книга уходит в заметки одним касанием.

## Перенос на ISKCON ONE LOVE

| Books | У нас | Что берём |
|---|---|---|
| Home — панель чтения | Практика ↔ Книги | дневная цель, серия, годовая цель, «продолжить» с процентом |
| Collections со счётчиками | Библиотека писаний | счётчик у каждой полки, режим правки с видимостью |
| Want to Read · Finished | путь ученика (Ц8) | статус чтения как первичная сущность, не тег |

| Разделы → категория → подкатегория | каталог писаний | три уровня, полки-карусели на нижнем |
| Нумерованный чарт | самое читаемое / цитируемое | у нас отсутствует |
| Suggest More / Suggest Less | подбор | правка выдачи из карточки |
| Line Guide | читалка стихов | удержание строки в длинной транслитерации |
| Тост с адресом | все действия | «добавлено **куда**», а не «готово» |
| Remove ≠ Remove Download | офлайн-модель | убрать из списка и стереть файл — разные действия |

| 🕳 | Чего нет |
