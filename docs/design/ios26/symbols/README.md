# Знаки Apple: SF Symbols

## Что здесь

`symbols.json` — векторные контуры символов Apple, извлечённые из пакета
`sf-symbols-lib` (npm). Инструмент: `tools/ios26-symbol.py`.

```
python3 tools/ios26-symbol.py --search walk      # найти знак по имени
python3 tools/ios26-symbol.py FigureWalk         # извлечь контуры
```

## Вектор или растровая маска

Проверено на шевроне «назад» кадра 3:

| Источник | расхождение зоны |
|---|---|
| растровая маска с кадра | **2.53 %** |
| вектор SF Symbols | 3.81 % |

Маска точнее, потому что снята с того же рендера, что и эталон: в ней уже сидят
хинтинг и сглаживание Apple. Вектор — идеальная форма, и она не совпадает с тем, как
система реально нарисовала знак на экране.

**Правило: в эталонах — маска с кадра, в приложении — вектор.** Вектор масштабируется
и не тащит артефакты съёмки, поэтому для живого интерфейса он и нужен; для сверки с
кадром он проигрывает.
