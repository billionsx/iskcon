# Токены: система переменных, собранная из замеров

Каждое значение здесь **пришло из замера**, а не из вкуса, и рядом стоит ссылка
на раздел `STANDARD_ios26_css.md`, где оно снято и чем. Токен без ссылки —
ошибка, его надо удалить или снять замером.

Единица — **точка (pt)**. Перевод в CSS: `1pt = 1px` при `@1x`, а вся геометрия
должна ложиться на **1/3 pt** (5.24: сетки в точках нет, есть физический пиксель
при @3x).

---

## 1. Поверхности

```css
--surface-canvas:      #000000;  /* холст                        5.19 */
--surface-card:        #1C1C1E;  /* карточка на холсте           5.19 */
--surface-card-media:  #1C1C1C;  /* медиа-карточка, на пиксель темнее */
--surface-sheet:       #1C1C1E;  /* холст листа                  5.16 */
--surface-on-sheet:    #2C2C2E;  /* карточка НА листе            5.16 */
--surface-menu:        #2C2C2E;  /* панель меню                  5.19 */
--surface-layer:       #181818;  /* материал плавающего слоя     5.19 */
--surface-player:      #111111;  /* панель плеера поверх видео   5.18 */
```

**Ступеней три, а не две.** Холст → карточка → карточка на листе. Лист сам
поверхность, и то, что лежит на нём, поднимается ещё на ступень.

## 2. Текст

```css
--label:            #FFFFFF;               /* основной            5.21 */
--label-secondary:  rgba(235,235,245,.60); /* ≈ #8E8E93           5.21 */
--label-tertiary:   rgba(235,235,245,.30); /* ≈ #48484A           5.21 */
```

**Серый с синим подтоном, а не нейтральный** (5.21): в замерах вторичного уровня
синий канал систематически выше остальных. `#8E8E8E` вместо `#8E8E93` даст
систематический сдвиг тепла по всему интерфейсу.

⚠️ `--label-tertiary` **не проходит по контрасту нигде** (5.27: 2.30 на холсте,
1.53 на листе). Это уровень выключенного и декоративного, не текст для чтения.

## 3. Акцент

Акцент принадлежит продукту, а не системе (2.6). Значения, снятые из хрома
десяти приложений, — палитра выбора, а не список для смешивания:

```css
/* синий по умолчанию — утилиты */
--accent-blue-dev:   #1896FC;   /* Developer              2.6 */
--accent-blue:       #4EA8F6;   /* Files, App Store       2.6 */
--accent-cyan:       #18AEFC;   /* TV                     2.6 */
/* продукты с собственной темой */
--accent-lime:       #C0FC18;   /* Fitness                2.6 */
--accent-pink:       #FC4860;   /* Music                  2.6 */
--accent-gold:       #D8AE3C;   /* Notes                  2.6 */
--accent-red:        #FC4E4E;   /* Calendar               2.6 */
```

Три системных цвета остаются **ролевыми** и акцентом не подменяются:

```css
--system-red:    #FF453A;   /* деструктив, «сегодня»   🍎 */
--system-green:  #30D158;   /* включённый тумблер      🍎 */
--system-orange: #FF9F0A;   /* предупреждение          🍎 */
```

## 4. Разделитель

```css
--separator:        #38383A;  /* центр семейства 0x31…0x41   5.33 */
--separator-width:  1pt;      /* НЕ 0.5 — полпикселя исчезает   2.4 */
```

Врезка разделителя — **следствие строки**, а не константа (5.19):

```css
--separator-inset-icon:  32pt;   /* под знаком              */
--separator-inset-text:  44pt;   /* под текстом без знака   */
--separator-inset-tile:  68.7pt; /* под плиткой иконки      */
```

## 5. Скругление

```css
--radius-card:        24pt;    /* карточка с врезкой 16     4.2 */
--radius-card-full:   37.8pt;  /* полноширинная медиа       4.2 */
--radius-capsule:     50%;     /* кнопка, таб-бар: h/2  5.23 */
```

**Показатель непрерывности n = 2.5–2.6** на девяти продуктах (4.2). CSS
`border-radius` рисует дугу; ближе к оригиналу — маска суперэллипса. Радиус
зависит от роли, показатель — нет.

## 6. Врезки и ширины

```css
--inset-card:      16pt;    /* карточка от края экрана    5.19 */
--inset-trailing:  16pt;    /* замыкающий элемент от кромки 5.24 */
--card-width:     361pt;    /* при ширине экрана 393       5.19 */
--card-full:      393pt;    /* полноширинная медиа         5.19 */
```

## 7. Ритм строки

```css
--row-height:          49pt;    /* обычная строка списка   5.19 */
--row-height-note:     53.3pt;  /* заголовок + дата + фрагмент */
--row-height-two-line: 73.7pt;  /* операция Wallet             */
--row-height-dense:    34.7pt;  /* строка в тонированной карточке */
```

## 8. Типографика

Замер — **cap-height**; кегль остаётся выводом и приблизителен (5.21).

```css
--cap-body:        10.33pt;  /* основная строка — 194 замера */
--cap-dense:       10.00pt;
--cap-secondary:    8.67pt;  /* вторичная подпись            */
--cap-section:     16.00pt;  /* заголовок раздела            */
--cap-title:       22.67pt;  /* крупный заголовок            */

--weight-regular:  400;  /* 59 % корпуса */
--weight-medium:   500;  /* 21 %         */
--weight-semibold: 600;  /* 10 %         */
--weight-bold:     700;  /* 10 %         */

--line-height:     1.30;    /* от кегля; = 1.84 × cap       5.28 */
--leading-body:   19pt;     /* при cap 10.33 — 65 замеров        */
--advance-char:  9.67pt;   /* средняя ширина знака = 0.94 × cap  5.37 */
```

Строка из N знаков занимает примерно **N × 0.94 × cap**. Русские интерфейсные
метки не длиннее английских (+5 % по сумме, 5.37) — проверять надо худшее слово
раздела, а не среднее.

**Выделение делается штрихом, а не размером** (5.21): cap 10.33 встречается
Regular, Medium и Bold. Пятого начертания в корпусе нет ни разу.

## 9. Знаки

```css
--glyph-menu:      14pt;    /* знак в строке меню       5.22 */
--glyph-small:     11.7pt;
--glyph-tile:      18pt;    /* плитка иконки в строке   5.22 */
--glyph-stroke:    1.33pt;  /* 4 физических пикселя     5.22 */
--glyph-stroke-sm: 1.00pt;  /* 3 пикселя                     */

--chevron-w:  5.67pt;  /* 17 px                        5.24 */
--chevron-h:  9.67pt;  /* 29 px, пропорция 1.71             */
--check-w:    6.7pt;   /* галочка — другой класс       5.22 */
--check-h:   10.0pt;   /* пропорция 2:3, не 1:1             */
```

**Штрих квантуется по пикселю** (5.22): только 3 или 4 физических пикселя,
промежуточных значений нет. Задавать долями нельзя.

## 10. Кнопка

```css
--button-height: 32pt;   /* постоянна: 18 замеров, разброс 0  5.23 */
--button-radius: 16pt;   /* = height / 2                          */
--button-fill:   #1C1C1C;
--button-hit-pad: 6pt;   /* до канона 44 pt              5.27 */

--chip-height: 35pt;     /* чип-фильтр — на ступень выше 5.30 */
--chip-radius: 17.5pt;
--chip-fill:   #2C2C2C;
```

**Высота постоянна, ширина по содержимому.** `--button-hit-pad` — прозрачный
отступ вокруг, а не увеличение капсулы: иначе она перестанет совпадать с
эталоном.

## 11. Плавающий нижний слой

```css
--layer-axis:      52pt;    /* середина над низом экрана   5.2 */
--layer-axis-player: 56pt;  /* поверх видео               5.18 */
--layer-h-tabbar:  62pt;
--layer-h-panel:   48pt;    /* одноэтажная панель, мини-плеер */
--layer-h-player:  38pt;
--layer-span:     351pt;    /* полоса: 21 + 351 + 21 = 393 5.20 */
--layer-gap:       12pt;    /* между капсулами            5.15 */
--layer-inset:     21pt;
```

**Постоянна ось, а не отступ снизу** (5.2). Нижняя кромка — следствие оси и
высоты детали: 62 → 21 pt от края, 48 → 28 pt.

**Слой имеет семь состояний** (5.20, 5.25): навигация · вход в раздел · плеер ·
перемотка · действие · второй этаж · **отсутствие** (при фокусе поиска).

## 12. Клавиатура и поиск

```css
--keyboard-top:    517pt;   /* верх при экране 852 pt  5.25 */
--keyboard-height: 335pt;
--search-lift:     133.3pt; /* подъём содержимого при фокусе */
```

⚠️ Фокус поиска **отнимает 126.7 pt видимой высоты**, несмотря на убранную
шапку (5.25). Список при фокусе короче, а не длиннее.

## 13. Кольца прогресса

```css
--ring-sm-radius:     6.7pt;   --ring-sm-stroke:  4.1pt;   /* 5.29 */
--ring-md-radius:    16.4pt;   --ring-md-stroke:  7.3pt;
--ring-lg-radius:    29.5pt;   --ring-lg-stroke: 16.8pt;
```

Отношение толщины к радиусу — не константа (0.45…0.61): крупное кольцо
относительно тоньше. Масштабировать одно из другого нельзя.

## 14. Сетка календаря

```css
--cal-col:      56pt;    /* = ширина экрана / 7, без врезок   5.31 */
--cal-row:      51pt;    /* строка сетки месяца                    */
--cal-row-list: 132.7pt; /* строка списочного вида                 */

--now-line:      2pt;      /* индикатор «сейчас»            5.32 */
--now-color:     #FC4446;  /* насыщеннее акцента Календаря       */
--now-capsule-w: 44.3pt;   --now-capsule-h: 16.3pt;

--event-bracket: 4pt;      /* цветная скоба блока события   5.35 */
--event-height: 28.3pt;
--empty-block-y: 440pt;    /* пустое состояние — чуть ниже середины 5.35 */
```

**Тени в тёмной теме нет** (5.35): на `#000000` темнее не бывает. Разделение
поверхностей — ступенью заливки, не `box-shadow`.

Индикатор принадлежит **дню**, а не экрану: линия занимает колонку дня
(15.7…228.7 pt), а не всю ширину.

У сетки недели врезок нет вовсе — иначе семь колонок не сойдутся. Это отличает
её от списка, у которого врезка 16 pt с каждой стороны.

## 15. Полки

```css
--shelf-pitch-tv:    112.5pt;  /* TV: и постеры, и портреты  5.26 · 5.32 */
--shelf-pitch-music: 153.0pt;  /* Music                      5.26 */
```

Шаг полки — **константа продукта, а не свойство содержимого**: TV раскладывает
постеры (112.3) и круглые портреты (112.7) по одной сетке.

🕳 Две полки — не закон. Общий шаг не выведен.

## Чего в этом наборе нет и почему

| Чего нет | Причина |
|---|---|
| Длительности и кривые переходов | 🕳 со статики не снимаются, нужна запись экрана |
| Светлая тема | 🕳 все десять продуктов сняты в тёмной |
| Dynamic Type кроме Large | 🕳 снят только Large |
| Тени и их параметры | ⏳ не измерены |
| Состояния кнопки (нажата, выключена) | 🕳 нет парных кадров |
| Заливка иконки активного таба | 🕳 нет парных кадров одного таба |

**Правило пополнения:** новый токен добавляется только вместе со ссылкой на
раздел, где снято его значение. Токен без замера — это вкус, выданный за
эталон, и он расползётся по коду как факт.
