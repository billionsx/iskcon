## 2026-06-12 — Слой ежедневной практики: подтверждение объёма

Прошлись по вопросу «чего не хватает, чтобы приложение стало ежедневным инструментом
преданного». Вывод: почти всё уже заложено в мастер-док. Ниже — что подтвердили,
какие уточнения объёма приняли и какие реальные пробелы зафиксировали.

### Подтверждено (уже в роадмапе — ничего не теряем)

| Область | Что | Где в роадмапе |
|---|---|---|
| Садхана | джапа-счётчик, дневник садханы, стрики, экадаши, праздники / дни ачарьев | M4 (П1 «Садхана»), M2 (календарь); arch разделы E–G |
| Ежедневный контакт | **стих дня + даршан дня** | M3 (стих), M4 (даршан); arch «крючки на каждый день» (G) |
| Новости ИСККОН (RU) | лента мировых новостей, переведённых на русский | arch: сущность `news`, маршрут `/news`, «крючок на каждый день» (G); этап — рядом с M5 |
| Прасад | система рецептов + молитвы предложения | arch раздел I («Прасад»), мини-апп «Рецепты» |
| Изучение | трекинг прочитанного / прослушанного (книги, главы, стихи, лекции) | M3 (библиотека), M9 (изучение писаний); arch `progress` |
| Рост | путь преданного, онбординг, офлайн | M7 (путь / геймификация), M1 (онбординг), arch «офлайн / PWA» |

### Решения и уточнения объёма (новое в этой сессии)

1. **Молитвы и мантры — без отдельного раздела.** Живут внутри «Киртаны / Бхаджаны»
   (M6): к каждой — текст + транслитерация + перевод, мелодия, режим «петь по слогам».
   Гаятри — только инициированным (гейт по профилю).
2. **Прасад = умная система рецептов.** Подбор по продуктам (что есть дома),
   по диете / ограничениям и т. п. — не просто плоский список рецептов.
3. **Ежедневный контакт = только стих дня + даршан.** Лекции относятся к медиа /
   изучению (M5 / M6), не к блоку ежедневного контакта.
4. **Изучение = трекинг + личная разметка.** Что пройдено (книги / главы / стихи /
   лекции) **плюс** закладки, выделения, заметки.

### Пробелы, которых ещё нет в мастер-доке (внести при следующей правке `PRODUCT_ARCHITECTURE.md`)

- **Закладки / выделения / заметки на стихах** — в архитектуре отсутствуют (grep: 0 упоминаний).
  Добавить в раздел «Изучение» + сущность пользовательской разметки (напр. `user_notes`).
- **Умный подбор рецептов по продуктам / диете** — рецепты в arch есть, «умного подбора»
  как функции нет. Уточнить в разделе I («Прасад»).

### Приоритет №1

**Счётчик джапы + дневник садханы** (M4). Ежедневная привычка возврата в приложение;
всё остальное наслаивается сверху.
