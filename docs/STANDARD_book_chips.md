# Стандарт факт-чипов книги

Чипы (`BookData.chips` в `apps/web/src/books.ts`) — три коротких факта на герой-карточке
книги. Источник истины — этот документ; ровно три чипа, в фиксированном порядке.

## Стандарт (книги со стихами)

```
n глав  /  n стихов  /  n лет
```

- **n глав** — общее число глав во всём произведении (не песней/лил).
- **n стихов** — общее число стихов (округлённо, в стиле «11 000+», «~18 000»).
- **n лет** — древность: сколько лет назад явлено/произнесено/составлено
  (традиционная датировка, округлённо, в стиле «5 000+ лет»).

## Стандарт (проза без стихов)

Для произведений, у которых нет стихов (изложения/сводные труды в прозе — например
«Нектар преданности»), третий чип заменяется на изначального автора:

```
n глав  /  n лет  /  изначальный автор
```

- **изначальный автор** — автор исходного санскритского труда, который изложен в книге
  (для «Нектара преданности» — Рупа Госвами, автор «Бхакти-расамрита-синдху»).
  Не путать с автором изложения (Шрила Прабхупада) — он указан в блоке автора.

## Текущие значения

| книга | чипы |
|-------|------|
| bg  (Бхагавад-гита)            | 18 глав · 700 стихов · 5 000+ лет |
| sb  (Шримад-Бхагаватам)        | 335 глав · ~18 000 стихов · 5 000+ лет |
| cc  (Шри Чайтанья-чаритамрита) | 62 главы · 11 000+ стихов · 500+ лет |
| brs (Нектар преданности)       | 51 глава · 500+ лет · Рупа Госвами |

Примечания к «n лет»:
- БГ / ШБ — традиционная датировка ~5 000 лет (эпоха Кришны / составление Вьясой
  в начале Кали-юги).
- ЧЧ / НП — эпоха Гаудия-вайшнавизма XVI века: Шри Чайтанья явился ~500+ лет назад,
  Рупа Госвами — Его современник. «500+ лет».

## Деление на песни/лилы

Число песней (ШБ) и лил (ЧЧ) в чипы НЕ выносится — оно показано в самом «Содержании»
(суб-табы) и в подзаголовках. Чипы держат единый формат глав/стихов/лет.
